OBS Studio Instructions

  1. Open WASD Vision
  2. Right-click the purple tray icon and select Options
  3. Choose your favorite color and preferred keyboard layout
  4. In OBS click the plus icon under Sources and select Window Capture
  5. Enter a name and click OK
  6. Select WASD Vision as the Window and click OK
  7. Right-click on your new source and select Filters
  8. Click the plus icon and select Color Key
  9. Enter a name and click OK
  10. Select Custom Color as the Key Color Type
  11. Click Select color and enter the HTML value from the table below 
               HTML Code
Turquoise  #00FF00
Orange     #DAA520
Purple     #7B68EE
Black      #000000
Pink       #FF00FF
Green      #00FF00
Yellow     #FFD700
White      #D3D3D3
  12. Click OK and then Close
  13. Repeat steps 4-12 with the Mouse Vision window if you would like to show the mouse overlay
  14. Enjoy!


OBS Classic Instructions

  1. Open WASD Vision
  2. Right-click the purple tray icon and select Options
  3. Choose your favorite color and preferred keyboard layout
  4. In OBS right-click under sources, select Add, then click Window Capture
  5. Select WASD Vision as the Window
  6. Check the Use Color Key option
  7. Double-click on the square next to Color and enter the Red, Green, and Blue values from the table below 
               Red  Green  Blue
Turquoise  0    255    0
Orange     218  165    32
Purple     123  104    238
Black      0    0      0
Pink       255  0      255
Green      0    255    0
Yellow     255  215    0
White      211  211    211
  8. Click OK twice
  9. Repeat steps 4-8 with the Mouse Vision window if you would like to show the mouse overlay
  10. Enjoy!
